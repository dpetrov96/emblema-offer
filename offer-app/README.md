# Emblema Offer App

Интерактивна оферта за Emblema App Platform - светла тема, Montserrat, glassmorphism.

## Стартиране

```bash
cd offer-app
npm install
npm run dev
```

Отвори [http://localhost:5173](http://localhost:5173)

## Export PDF

Натисни **Export PDF** в sidebar-а или `Cmd+P` → Save as PDF.
