1. Обхват на проекта

Включено

  • Backend REST API + PostgreSQL
  • Multi-tenancy (множество сгради)
  • Auth + RBAC (4 роли) + 2FA
  • Mobile app - iOS & Android
  • Web Admin (Manager + Central Admin)
  • Новини, Събития, Спорт, Лоялност, Радио
  • Access management + автомобилни номера
  • Push + in-app + email нотификации
  • Audit log + CI/CD + QA + App Store
  • External Smart Home links (не вътрешна интеграция)

Извън обхват

  • Smart Home модул (врати, осветление)
  • Домофон в приложението
  • LPR / разпознаване на номера
  • Hikvision / Shelly интеграция
  • Marketplace / плащания
  • Hosting разходи (~50–150 €/мес.)
Забележка: Часовете са калибрирани за AI-assisted разработка (Cursor + Claude) - ~30–35% по-бързо от класическо ръчно писане на код. QA, UAT и App Store submission не се намаляват пропорционално.
